2. To add data for a new photo item

- Endpoint: http://localhost:3000/submit (NO parameter and JSON.stringified form data required including image, author, alt, tags and description)

- HTTP Method: POST (Error: status 500, Success: 201)

- MIME format in response: NO response data

- Example:

input(

{"author":"Tim Berners-Lee",

"alt":"Image of Berners-Lee",

"tags":"html,http,url,cern,mit",

"image":"https://upload.wikimedia.org/wikipedia/commons/9/9d/Sir_Tim_Berners-Lee.jpg",

"description":"The internet and the Web aren't the same thing."})

output()


3. To list the data of a specific item

- Endpoint: http://localhost:3000/receive/:id (YES parameter(ID of the item) and JSON.stringified form data required including ID)

- HTTP Method: GET (Error: status 500, Success: 200OK)

- MIME format in response: application/json

- Example:

input({"id":3})

output(

{"id":1,

"author":"Tim Berners-Lee",

"alt":"Image of Berners-Lee",

"tags":"html,http,url,cern,mit",

"image":"https://upload.wikimedia.org/wikipedia/commons/9/9d/Sir_Tim_Berners-Lee.jpg",

"description":"The internet and the Web aren't the same thing."})


4. To change data of a specific item

- Endpoint: http://localhost:3000/item/:id (YES parameter(ID of the item) and JSON.stringified form data required including ID, url, author, alt, tags and description)

- HTTP Method: PUT (Error: status 500, Success: 200OK)

- MIME format in response: NO response data

- Example:

input(

{"id":1,

"author":"Tim Berners-Lee",

"alt":"Image of Berners-Lee",

"tags":"html,http,url,cern,mit",

"image":"https://upload.wikimedia.org/wikipedia/commons/9/9d/Sir_Tim_Berners-Lee.jpg",

"description":"The internet and the Web aren't the same thing."})

output()


5. To remove data of a specific item

- Endpoint: http://localhost:3000/delete/:id (YES parameter(ID of the item) and JSON.stringified form data required including ID)

- HTTP Method: DELETE (Error: status 500, Success: 204)

- MIME format in response: NO response data

- Example:

input({"id":3})

output()


6. To reset the database

- Endpoint: http://localhost:3000/reset (NO parameter and NO data need to be sent)

- HTTP Method: DELETE (Error: status 500, Success: 200OK)

- MIME format in response: NO response data

input()

output()
